Team Task Manager

Description:
This is a full-stack task management application developed using React, Node.js, Express, and MongoDB. The system allows users to register, log in, and manage tasks efficiently.

Features:
- User authentication (Login and Register)
- Create, update, and delete tasks
- Task status tracking (Todo, In Progress, Done)
- Task assignment and project categorization
- Dashboard showing task statistics

Technologies Used:
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

Setup Instructions:

Backend:
1. Navigate to backend folder
2. Run: npm install
3. Create .env file with:
   MONGO_URI=your_mongodb_url
   JWT_SECRET=your_secret_key
4. Run: node server.js

Frontend:
1. Navigate to frontend folder
2. Run: npm install
3. Run: npm start
